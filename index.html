About me

My journey into software development began through teaching myself coding skills using resources online. Having started learning Python independently, I enjoyed using logic to write code and using the problem solving skills that I had already developed in my previous career as a scientist.

Quickly realising that I would enjoy continuing my development with a new career in tech, I enrolled on a Full Stack Software Engineer course, learning to use Javascript, React.js, Python and other technologies. I am now looking to use, and continue to develop my skills in a role that will challenge me as a junior developer.

Projects

    A full stack app built using MERN technology, with CRUD functionality. As a group we created an app that could be used to buy and discuss drinks. I worked across the stack, but primarily focussed on the comment and rating systems and authentication.

    In two days my partner and I paircoded a News service app using React.js and a public API. We focussed on simple, clear design and responsiveness across devices. I challenged myself by building a feature that would allow users to save articles.

    Using JavaScript, HTML and CSS I built a grid-based game. This modern take on pacman allowed me to understand key concepts in JavaScript and I particularly enjoyed the challenge of using logic to code intelligent movement for the pacman ghosts.
